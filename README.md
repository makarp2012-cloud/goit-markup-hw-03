## 📁 Структура проєкту
Коректна структура папок та файлів:
```text
goit-markup-hw-02/
├── css/
│   ├── common.css
│   ├── header.css
│   ├── hero.css
│   ├── features.css
│   ├── team.css
│   ├── portfolio.css
│   ├── footer.css
│   └── main.css
├── images/
│   └── (зображення з макета)
└── index.html

```

## 🛠 Технічні вимоги

* **CSS:** Використання `modern-normalize`, імпорт модулів у `main.css`, відсутність `!important`.
* **HTML:** Семантична розмітка, валідність коду, використання списків `<ul>` для однотипних елементів.
* **Шрифти:** Одночасне підключення *Raleway* та *Roboto* через Google Fonts.
* **Якість:** Форматування коду за допомогою **Prettier**, відсутність кирилиці/пробілів у назвах файлів.
* **Стандарти:** Дотримання настанови (Best Practices), правильна робота з успадкуванням шрифтів та кольорів.
